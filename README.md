# Personal Portfolio Website

## Description

This is a responsive personal portfolio website created to showcase my skills, projects, and experience as a Web Development Intern and Computer Science student. The website features a modern blueprint-inspired design with smooth navigation and a clean user interface.

---

## Features

* Responsive portfolio website
* Modern UI design with blueprint-style background
* Smooth scrolling navigation
* About section with personal information
* Skills showcase section
* Projects display section
* Contact section with GitHub and email links
* Mobile-friendly layout

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (basic concepts)
* Bootstrap
* Google Fonts
